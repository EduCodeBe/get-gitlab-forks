# Get Gitlab Forks

Do you want to retrieve all the forks of your gitlab project you have
access to? If so, this script could help you.

To learn how to use it:

```
./get-gitlab-forks.sh -h
```

## Dependencies

You will need:

- bash >= 5
- jq
- curl
